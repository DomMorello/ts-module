## 🚀 Getting Started

> 일반적인 TypeScript 사용이 아닌 더 비범한 타이핑을 위해 작성합니다.

✔️ 모듈을 고려하여 타이핑합니다.  
✔️ 사용될 곳을 위한 타이핑을 합니다.  
✔️ 근사한 로직 구현보다는 타입 선언과 테스트에 초점으로 둡니다.  
✔️ (타입 선언 => 테스트 케이스 => 구현)을 우선 순위로 훈련합니다.
